## 个人网站搭建

网站部署在 digitalocean 的 ubuntu 服务器上.

前端项目使用 nginx 作为服务器,node 服务使用 PM2 管理

每个项目都跑在独立的 docker container 中

通过 github action 配合 self-hosted runner 监听 git push main 自动重新构建运行 docker.以此实现自动化部署

使用 nginx 的 proxy_pass 转发请求到对应端口的应用.

打包的带 hash 的静态资源配置长缓存...
