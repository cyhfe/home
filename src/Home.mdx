## analytics

简单的网页分析工具.由 3 个部分组成

- 目标网站插入 script,监听路由变化和页面可见性发送统计数据

- express + prisma + sqlite 持久化数据

- 前端数据图表展示

- 自动化部署: github actions + docker compose
